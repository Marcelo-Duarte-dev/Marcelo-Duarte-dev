Que os jogos comecem

O que aprendi por enquanto:


Aprendi que a tag p se faz um parágrafo e utilizamos o espaço quando se dá parágrafo apenas quando literalmente for escrever outro parágrafo, caso contrário uso a tag br ou css (que ainda vou aprender)

Além disso, aprendi algumas tags para formatação de texto como a tag i, para deixar em itálico como o u que siginifica underline para deixar literalmente uma linha abaixo do texto e sublinhar texto. Aprendi a marcar o texto com a tag mark texto marcado, a tag b para deixar o texto em negrito Texto em negrito, ou a tag strong que também serve para colocar um texto em negrito, mas têm a função de dizer para o navegador que aquele texto é relevante Texto em negrito strong.

Apredi a utilizar a tag sub que serve para texto ou número abaixo 2 ou sup para colocar o texto ou número acima 2. Por último gostaria de dizer que esqueci de duas tags kkkk, mas todas as tags anteriores consegui relembrar, as tags que eu esqueci são a del e small, estava me perguntando qual era a tag para sublinhar o centro do texto, e é exatamente a del, Texto com a tag del. A tag small, ela serve para deixar as letras pequenas letra pequena.
